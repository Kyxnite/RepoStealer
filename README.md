# Ruby Source Code Stealer
A Fast, Simple Source Code Stealer, Coded in RUBY



## Features

- **Steal Source Code**
    - Scan Through Repos Folder
    - Zip Up Repositories 
    - Copy all stolen files
    - Paste them into new Directory
    - Send to Discord Webhook

  
     
## Installation

```ruby
gem 'rest-client'
```
## Usage/Examples


```python 
ruby Stealer.rb
```
